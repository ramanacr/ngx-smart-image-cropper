# ngx-smart-image-cropper - Product Requirements

## Vision
Build the best Angular 21-native image cropper library.

## Goals
- Angular 21 standalone-first
- Signals-based state management
- SSR compatible
- Accessibility compliant
- Mobile touch support
- High performance canvas rendering

## Core Features
- Rectangle crop
- Circle crop
- Aspect ratio lock
- Zoom, pan, rotate, flip
- Blob/File/Base64 export
- Keyboard support
- Undo/Redo

## Non-Functional Requirements
- >90% test coverage
- Tree-shakeable
- MIT license
- Semantic versioning
